# 📊 Dashboard de Vendas do Xbox com Excel

Este projeto apresenta um **Dashboard interativo de vendas do Xbox**, desenvolvido no **Microsoft Excel**, com o objetivo de analisar e visualizar dados de forma clara e dinâmica.  

O dashboard permite acompanhar métricas de vendas, identificar tendências e apoiar a tomada de decisão com base em dados.

---

## 🚀 Funcionalidades

- Visualização interativa das vendas do Xbox  
- Gráficos dinâmicos para análise de desempenho  
- Segmentação por período, região e categoria de produto  
- Indicadores de desempenho (KPIs) para insights rápidos  
- Interface amigável e organizada para facilitar a interpretação dos dados  

---

## 🛠️ Tecnologias Utilizadas

- **Microsoft Excel**  
  - Tabelas dinâmicas  
  - Gráficos dinâmicos  
  - Segmentação de dados  
  - Formatação condicional

## 📈 Insights Possíveis

- Comparação de vendas por período

## 🔧 Como Usar

1. Baixe o arquivo `Dashboard.xlsx`  
2. Abra no **Microsoft Excel** (versão 2016 ou superior recomendada)  
3. Utilize os filtros e segmentações para explorar os dados  

---

## 📌 Objetivo do Projeto

Este projeto foi desenvolvido para fins de **aprendizado e prática em análise de dados com Excel**, demonstrando como criar dashboards profissionais e interativos sem necessidade de ferramentas avançadas de BI.
